# fleetboom
透明全屏 Tkinter 小游戏：鼠标追捕逃逸飞船，科幻线稿飞船+多种子弹特效+漂浮行星背景，支持难度 1~11 与实时缩放。
A fullscreen transparent Tkinter mini-game: chase an evasive spaceship with your mouse. Sci-fi line ships, multi-style bullet VFX, drifting planets, difficulty 1–11, and live scaling.
# 舰队爆炸！！！（Fleet Boom）

一个用 **Python + Tkinter Canvas** 做的透明全屏小型“追捕游戏/屏保”：
你用鼠标指针追击会逃逸的飞船，贴住一小段时间即可触发爆炸计数。  
包含多种飞船外形、科幻子弹特效、漂浮行星背景，并支持难度与实时缩放。

## 特色
- 🛰️ 多种飞船外形（线稿风格）+ 尾焰动画
- ✨ 多种子弹视觉特效（激光/等离子/星火/波动/碎片）
- 🪐 漂浮行星背景（陨坑/星环/卫星），永远缓慢随机漂移
- 🎮 游戏模式 & 屏保模式可切换
- 🔥 难度 1–11（Lv.11 需要预判走位）
- 🎚️ 控制面板：实时切换透明背景、调整飞船/星球大小、难度/自动升级等

## 操作
- 鼠标靠近飞船：飞船会逃跑
- 鼠标贴近并保持短暂时间：飞船爆炸（计数 + 可自动升级）
- ESC：退出全屏叠加层
